# Lucrare de laborator 4: Publicarea cu GitHub Pages

## Obiective

- Înțelegerea conceptului de GitHub Pages.
- Publicarea unui site web static direct dintr-un repository GitHub.
- Personalizarea unei pagini HTML și accesarea acesteia online.

## Realizați următoarele sarcini:

1. Creează un nou repository GitHub numit laborator4. Clonează-l local.

2. Crează o pagină HTML - un singur fișier `index.html` cu un mesaj simplu.
   Încarcă fișierul în repozitoriu.

3. Activează GitHub Pages.

4. Accesează pagina publică. Accesează linkul generat de GitHub Pages pentru a vedea site-ul publicat.

## Extensie opțională

5. Creează un fișier `style.css` și leagă-l în `index.html`. Adaugă o imagine în pagină.
   Experimentează cu stiluri și layout-uri CSS.

6. Creează un workflow care va automatiza procesul de preluare și publicare pe GitHub Pages a paginii create.
